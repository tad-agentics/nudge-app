- Install prompt: add `app/hooks/useInstallPrompt.ts` (if missing).
- Global: `QueryClientProvider`, Supabase session provider, theme from Make.

### Acceptance

- `npm run build` green; prerender `/` includes title/description/OG; RLS blocks cross-user selects in a smoke test.

## Wave 3 — Core engine & Do next (`core-engine`)

### Backend

- `tasks` table RLS; active task count check (≤5 free) in Edge or RPC before insert.
- `llm-engine`: `structure`, `reason_batch`, `decompose`, `weekly_insight` actions; output schemas + validation gate (TypeScript shared if possible).
- `behavioral_events`: insert on surfaced/completed/skipped per northstar §3d.

### Frontend

- **Make source:** `DoNextScreen.tsx`  
- **Components (from spec):** `TaskCaptureInput`, `DoNextCard` (4px top stripe if `skip_count >= 3`), `TaskRationale`, `ScheduledTimeChip`, `StartButton`, `DoneButton`, `SkipButton`, `TaskInboxList`, `DecompositionModal`, `ConnectGoogleModal`, `FreeLimitModal`, `SkipToast`, recurring + NL edit + push prompt (per screen spec deltas).  
- **Data vars:** `tasks.*`, `profile.*`, `limits.*` — hooks in `app/hooks/`.  
- **Copy slots:** `input_placeholder`, `section_do_next`, `task_done*`, `skip_toast`, `auth_prompt_*`, `decomposition_offer`, `free_limit_body`, `nav_*`, `recurring_created`, `edit_confirm_*`, `push_permission_rationale`.  
- **States:** loading (structuring inline), error, earned empty `no_tasks`, first_open placeholder.  
- **Dopamine:** **D1, D2, D4** on Done sequences (EDS §6) — `motion` + 1.5–2.5s pause.

### Acceptance

- Capture → structure → reason → top task; Done/Skip update DB + skip stripe UI; freemium modal at 5 actives.
